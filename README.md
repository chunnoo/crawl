# Crawl

### Setup
Change the constans from line 5 to line 10 in *main.js*.

`phpsessid` and `simpleSAMLAuthToken` has to be set to your valid cookies from *tp.uio.no/ntnu/rombestilling*

### Running the code
Install dependencies
```
npm install
```
Run
```
node main.js
```
